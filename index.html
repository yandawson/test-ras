Theory JavaScript

In the last assignment, you created a static web site. A dynamic web site is responsive to user input. In this lab, we will add client-side dynamism to your site. In a later lab, we will add server-side dynamism. JavaScript is used everywhere throughout the modern web to respond to user input, perform user validation, load third-party content, and more. Have a look at the page source of a typical web page, and note how frequently JavaScript is invoked and for what purpose.
